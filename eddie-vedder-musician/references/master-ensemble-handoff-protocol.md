# Grand Master Ensemble Handoff Protocol

Six-persona prompt channel for the **skit-dance master personalities**:

| Skill | Persona | Role |
|-------|---------|------|
| `andy-warhol-director` | Andy Warhol Director | Screen tests, cast, repetition grid, routing |
| `dr-seuss-script-writer` | Ted Rhymewell | Rhyming scripts, moral, dance beats in verse |
| `stephen-spielberg-producer` | Steven Reelwright | Blockbuster treatment, set pieces, greenlight |
| `le-corbusier-set-designer` | Charles-Édouard Modulier | Modulor sets, zones, spatial grid |
| `eddie-vedder-musician` | Edward Stonevoice | Soundtrack, vocals, cue map, Voxtral blocks |
| `tom-cruise-stuntman-personality` | Marcus "Mav" Freefall | Practical stunts, one-take blocking, safety meeting |

Sub-protocols: Warhol↔Seuss · triple (Spielberg) · quadruple (Corbu) — see paths at bottom.

**Quintet mode:** Omit Mav — same order without steps 3 and 11 below (*"all five"*, *"no stunts"*).

---

## Full Stack Order (All Six)

```
User idea
    ↓
1. Steve  — SPIELBERG PRODUCER OUT (+ SPIELBERG RHYME BRIEF if rhyming)
2. Corbu  — CORBU SET DESIGN OUT (+ CORBU RHYME GEOGRAPHY)
3. Mav    — MAV STUNT COORD OUT (zones + set pieces → practical gags)
4. Eddie  — VEDDER SOUNDTRACK OUT (scores zones + stunts + grid)
5. Andy   — WARHOL PROMPT IN (embeds all refs)
6. Ted    — SEUSS PROMPT OUT (may weave Eddie's hook meter + Mav beats)
7. Andy   — SEUSS PROMPT ACK + Factory brief
8. Steve  — SPIELBERG GREENLIGHT ACK
9. Corbu  — CORBU SET SIGNOFF
10. Eddie — VEDDER SESSION SIGNOFF
11. Mav   — MAV STUNT SIGNOFF
12. video-creator P2/P6 + Voxtral + Seedance action cues
```
