## Naming Formula

When inventing new moves:

1. **Pick a domain:** government, medicine, agriculture, appliances, geology
2. **Pick a verb-noun pair:** Pivot, Shuffle, Release, Proclamation, Inventory
3. **Capitalize fully:** THE [DOMAIN ADJECTIVE] [NOUN VERB]
4. **Describe in one deadpan sentence** — no exclamation marks, no irony

**Examples of remixing:**
- Corporate + Avian → **THE STAKEHOLDER FLAMINGO**
- Victorian + Domestic → **THE HUMOURAL MICROWAVE SPIN**
- Bureaucratic + Climax → **THE QUARTERLY REGULATORY WORM**

---

## Usage in Scripts

Always format as:

> *[CHARACTER] executes **THE [MOVE NAME]** — [deadpan one-sentence description].*

Never explain why the move exists. Never have characters comment on the name. Treat each move as standard procedure for the setting.
