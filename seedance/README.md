# Seedance 2.0 Prompt Writing Skill

English and Chinese agent skills for writing effective video generation prompts for [Jimeng Seedance 2.0](https://jimeng.jianying.com/), ByteDance's multimodal AI video generation model.

**Upstream:** [dexhunter/seedance2-skill](https://github.com/dexhunter/seedance2-skill) (MIT)

## Install (Grok)

```powershell
Copy-Item seedance\SKILL.md "$env:USERPROFILE\.grok\skills\seedance-prompt-en\SKILL.md"
Copy-Item seedance\zh\SKILL.md "$env:USERPROFILE\.grok\skills\seedance-prompt-zh\SKILL.md"
```

## Files

| File | Language | Description |
|------|----------|-------------|
| `SKILL.md` | English | Prompt writing guide for Seedance 2.0 |
| `zh/SKILL.md` | 中文 | Seedance 2.0 提示词撰写指南 |
